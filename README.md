# QRB ROS Interfaces

## Overview

qrb_ros_interfaces is a ros2 package contains the custom qrb ros messages.

## System Requirements

- ROS 2 Jazzy.

## Supported Platforms

This package is designed and tested to be compatible with ROS 2 Humble running on Qualcom RB3 gen2.

| Hardware                        | Software                |
| ------------------------------- | ----------------------- |
| RB3 gen2                        | LE.QCROBOTICS.1.0       |
